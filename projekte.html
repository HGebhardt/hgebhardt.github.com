    SpotJotter

    • Status: development 
    • spotjotter.herokuapp.com
    SpotJotter bietet seinem Benutzer die Möglichkeit ortsgebundener Notizen auf Basis von Foursquare-Venues. Bei einem Check-In sendet Foursquare eine Push-Benachrichtigung an SpotJotter, welcher eine Auswertung des aktuellen Benutzerstandortes ermöglicht. Sofern in der Datenbank „Jottings“ für das bezogene Foursquare-Venue hinterlegt sind, wird eine Antwort an Foursquare gesendet, so dass der Benutzer eine Benachrichtigung innerhalb der Foursquare-Applikation erhält. Der Dienst ermöglicht somit benutzerdefinierte, ortsbasierte Erinnerungen, die bei Bedarf angezeigt werden.
    Beim Anlegen von neuen Notizen ist eine Suche gegen die Foursquare-Venue-Datenbank möglich. Um einen möglichst performanten Betrieb zu gewährleisten und nicht die Limits der Foursquare-API auszureizen, wird als Cache eine Memcached-Lösung eingesetzt, die vorhandene Suchergebnisse eine bestimmte Zeit lang zwischenspeichert.

    Technologien:

    • Ruby on Rails
    • PostgreSQL
    • Memcached
    • foursquare API

    quetch (foursquare Hackathon 2013)

    • Status: proof of concept 
    • quetch (Foursquare Hackathon 2013)
    • quetch.herokuapp.com
    Die Anwendung „quetch“ ist das Ergebnis einer Teamarbeit im Rahmen des Foursquare-Hackathons 2013. Sie erlaubt Benutzern, Aufgaben zu stellen und Freunde herauszufordern. Die Aufgaben sind jeweils an eine konkrete Ortsangabe und an eine bestimmte Kategorie gebunden. Anhand von Check-Ins auf Foursquare wird der aktuelle Benutzerstandort bestimmt, so dass umgebungsrelevante Aufgaben von Freunden ermittelt werden können. Der Aufgabensteller ist immer auf dem aktuellen Stand, da die Aufgaben durch Benutzerinteraktionen aktualisiert werden immer einem bestimmten Status vorweisen.
    Die Verwendung von Sidekiq in Kombination mit Redis erlaubt die Ausführung von ressourcenintensiven Funktionalitäten im Hintergrund mit loser Kopplung zur eigentlichen Anwendung.

    Technologien:

    • Ruby on Rails
    • PostgreSQL
    • Redis
    • foursquare API

    formbot

    • Status: development 
    • formbot.herokuapp.com
    Mit Hilfe der Anwendung formbot lässt sich die Einbindung von Kontaktformularen in Websites vereinfachen. Statt eine serverseitige Mail-Komponente zu integrieren, bietet der Dienst eine simple Möglichkeit, die Formulardaten zu verarbeiten und über definierbare Vorlagen an eine bestimmte E-Mail-Adresse weiterzuleiten. Die Verwendung der Liquid-Markup-Sprache bietet dabei vielfältige Möglichkeiten bei der Verarbeitung der Daten.

    Technologien:

    • Ruby on Rails
    • PostgreSQL
    • Liquid Markup
    • Mailgun API

    erweiterungen.de

    • Status: discontinued 
    • erweiterungen.de
    Das Portal erweiterungen.de war über eine Zeit von fünf Jahren die zentrale Anlaufstelle für deutschsprachige Erweiterungen für Mozilla-basierte Anwendungen. Als einfache Liste gestartet, wurden dem Projekt schnell immer mehr und immer komplexere Aufgaben zuteil. Insbesondere das Management von verschiedenen Versionen hinsichtlich der Kompatibilität mit Anwendungsversionen und deren Aktualisierungen war eine besondere Herausforderung in diesem Projekt. Aufgrund der hohen Zugriffszahlen bedurfte es weiterhin ausgefeilten Caching-Strategien, um die Serverlast möglichst gering zu halten und dennoch eine anwenderfreundliche Umgebung bereitzustellen.

    Technologien:

    • PHP-basiert
    • Smarty (PHP Template Engine)
    • MySQL
